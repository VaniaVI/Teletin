- Información general sobre Teletón.
- Preguntas sobre la Campaña anual.

- Configura el bot para acceder y utilizar esta información.

Desafío 1: Desarrollar un ChatBOT
Paso 1: Acceso y Configuración Inicial
Inicia sesión en tu cuenta de Automation Anywhere.

Crea un Nuevo Bot:

En la sección de Bots, selecciona "Crear Nuevo Bot".

Paso 2: Diseño del Flujo de Conversación
Define las Categorías de Información:

Información general sobre Teletón (misión, visión, historia).

Preguntas sobre la Campaña anual.

Información general sobre diagnósticos y tratamientos.

Crear Intenciones y Entidades:

Intenciones: Estos son los diferentes tipos de consultas que el bot puede entender. Por ejemplo, "Información sobre Teletón", "Detalles de la campaña", "Información de tratamientos".

Entidades: Estos son elementos específicos que el bot puede reconocer dentro de una intención. Por ejemplo, "misión", "visión", "fecha de campaña".

Diseña el Diálogo:

Utiliza el editor de flujos de trabajo para arrastrar y soltar diferentes comandos que representen las respuestas del bot.

Paso 3: Configurar el Repositorio de Información
Cargar Información:

Asegúrate de que toda la información necesaria esté disponible en un repositorio accesible para el bot.

Configura el Acceso al Repositorio:

Asegúrate de que el bot pueda acceder y recuperar información del repositorio según sea necesario.

Paso 4: Prueba y Despliegue
Prueba el Bot:

Ejecuta el bot en modo de prueba para verificar que responde correctamente a las consultas.

Despliega el Bot:

Despliega el bot en una página web simulada para que esté accesible a los usuarios.

Desafío 2: Desarrollar un Bot de Agendamiento de Horas
Paso 1: Configuración Inicial
Crea un Nuevo Bot:

En la sección de Bots, selecciona "Crear Nuevo Bot".

Paso 2: Diseñar el Proceso de Agendamiento
Define el Flujo de Trabajo:

Recopilación de Datos del Paciente: Captura la información necesaria del paciente.

Verificación de Disponibilidad: Consulta la disponibilidad de horarios y días.

Confirmación de Cita: Proceso para confirmar y notificar la cita al paciente.

Configurar Interacciones:

Diseña las interacciones del bot para que sea intuitivo y fácil de usar.

Paso 3: Integración de Material Complementario
Incorporar el Modelo de Atención Teletón 2024:

Asegúrate de que el bot siga las directrices del Modelo de Atención y el Reglamento asistencial.

Paso 4: Prueba y Despliegue
Prueba el Bot:

Realiza pruebas exhaustivas para asegurar que el bot maneja correctamente las citas.

Despliega el Bot:

Despliega el bot en una página web simulada para que los usuarios puedan interactuar con él.

Crecimiento y Escalabilidad
Diseño Modular para la Versión 2.0
Planifica la Integración Futura:

Diseña el bot de manera que sea fácil de integrar con el sistema de agendamiento de Teletón en una futura versión.

Preparar para la Escalabilidad:

Asegúrate de que el bot pueda escalar para manejar un mayor volumen de usuarios y consultas.

Presentación del Proyecto
Preparación de la Presentación:

Destaca la originalidad, utilidad y creatividad del bot.

Explica el proceso de desarrollo y cómo se llegó a la solución final.

Realiza una demostración en vivo de ambos bots.

Ejemplo de Uso
Ejemplo del ChatBOT:
Usuario: "¿Cuál es la misión de Teletón?" Bot: "La misión de Teletón es... [respuesta detallada]."

Ejemplo del Bot de Agendamiento:
Usuario: "Quiero agendar una cita para mi hijo." Bot: "Claro, ¿podrías proporcionarme el nombre completo del paciente y una fecha preferida?"
